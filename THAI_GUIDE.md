# THAI_GUIDE.md — Тай-формат (Thai Mixed / MM / WW)

Короткая инструкция по работе с тай-сессией в UI: `formats/thai/thai.html`.

## Запуск

Открывайте страницу формата с параметрами:

- `mode`:
  - `MF` — Микст (М/Ж)
  - `MM` — Мужской
  - `WW` — Женский
- `n` — количество игр (поддерживаются `8` и `10` на текущем генераторе)
- `seed` — seed генерации расписания
- `trnId` — идентификатор турнира (используется для ключей хранения сессии)

Пример:
`formats/thai/thai.html?mode=MF&n=8&seed=1&trnId=thai_123`

## Ростер (выбор игроков)

1. Выберите нужных игроков в блоках:
   - для `MF` — отдельно мужчины и женщины
   - для `MM`/`WW` — один пул
2. Нажмите `📋 Автобаланс` (рекомендуется).
3. После того как чекбоксы заполнятся до требуемого количества, станет доступна кнопка:
   - `✅ Запустить сессию`

## R1 — экран кортов (ввод счетов)

После старта вы попадете на панель кортов (R1):

1. Для каждого корта введите счет для команд:
   - `own` (левая сторона карты)
   - `opp` (правая сторона карты)
   Кнопки `− / +` меняют значение на 1.
2. Zero-Sum бар (`thai-zs-bar`) показывает состояние тура:
   - `Zero-Sum OK — тур сбалансирован` (и тур можно завершать)
   - либо предупреждение/ошибка, пока тур не сбалансирован.
3. Переключатель отображения (F1.6):
   - `Счёт` — показываются `diff/pts` бейджи
   - `Diff` — вместо счета показывается большой `diff`
4. Бейджи отдыха (F1.5):
   - показывают игроков, которые отдыхают в текущем туре (кто не участвует в матчах тура).

Дальше:

- `📊 Таблица` — кросс-таблица standings для активного тура.
- `▶ Следующий тур` — переход к следующему туру (включается, когда Zero-Sum OK и все счета введены).

## R2 — посев (seeding)

На экране standings нажмите:

- `🎯 Посев R2`

Откроется экран зон R2 с карточками:

- `🔴 Hard`
- `🟠 Advance`
- `🔵 Medium`
- `🟢 Lite`

Внутри каждой зоны отображаются:

- игроки
- их итоговые значения за R1 (PTS/Diff/Wins/K)

После проверки зон нажмите:

- `▶ Играть R2`

## R2 — игры

На экране R2 карты кортов переиспользуют формат R1, но с цветовой разметкой зон.

1. Вводите `own/opp` счет так же, как в R1.
2. Внизу панели доступна кнопка:
   - `🏆 Завершить`

## FINISHED — победители, номинации, Telegram-отчет

После `🏆 Завершить` показывается:

- подиум (топ-3) и итоговая таблица standings
- блок `Номинации` (6 алгоритмов)
- блок `Telegram-отчёт`:
  - автоматически заполненный текст
  - кнопка `📋 Скопировать в буфер`

## Что считаем в формулах

Текущая математика позаимствована из контракта тай-формата:

- точки (`pts`) считаются от `diff` по тай-правилам (`thaiCalcPoints`)
- `K` — коэффициент от суммы diff (`thaiCalcK`)
- Zero-Sum тура проверяется суммой diff = 0
- standings/nomination ранжируются по wins/diff/pts/K (с тай-брейками)
